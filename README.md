# Démarrage de MAN'S LOCATION

Ce projet a été initialisé avec [Create React App](https://github.com/facebook/create-react-app).

**Note: Gardez les commandes npm en anglais pour compatibilité.**

## Scripts disponibles

Dans le répertoire du projet, vous pouvez exécuter :

### `npm start`

Lance l'application en mode développement.\
Ouvrez [http://localhost:3000](http://localhost:3000) dans votre navigateur.

La page se rechargera automatiquement lors des modifications.\
Les erreurs de lint apparaîtront dans la console.

### `npm test`

Lance le testeur en mode interactif surveillance.\
Consultez la section [running tests](https://facebook.github.io/create-react-app/docs/running-tests) pour plus d'infos.

### `npm run build`

Compile l'app pour production dans le dossier `build`.\
Optimise React en mode production.

Le build est minifié avec hashes dans les noms.\
L'app est prête au déploiement !

Voir [deployment](https://facebook.github.io/create-react-app/docs/deployment).

### `npm run eject`

**Attention : opération irréversible ! Une fois `eject` exécuté, pas de retour en arrière.**

Si les outils de build ne vous conviennent pas, `eject` anytime. Supprime la dépendance build unique.

Copie configs (webpack, Babel, ESLint...) dans votre projet pour contrôle total. Tous les scripts sauf `eject` marchent toujours.

Vous n'avez pas à utiliser `eject`. Parfait pour petits/moyens déploiements. Personnalisez quand prêt.
